# Animating Unitree G1 upper body in A-Frame

## Installation and Startup
```
pnpm install
```
```
pnpm build
```
```
pnpm dev
```
or
```
cd dist/ && <START HTTPS SERVER>
```
## How to Operate this Sample Scene

1. Pushing the thumbstick and selecting "(pie menu)ray" toggles the ray on
   and off.
2. Pushing the thumbstick and selecting "g1r" for G1 right arm and "g1l"
   for G1 left arm.
3. When the ray is displayed, the arms can be moved during grip down.
4. To close the hand, push A or X button, and to open, B or Y.
